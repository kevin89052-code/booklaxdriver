# BookLAXDriver 上线说明

## 你现在拥有的版本

这是 `booklaxdriver.com` 的网址版预约平台，也可以下载安装到手机桌面，外观接近 App。

客户可以：

- 查看 LAX 私人司机服务卖点
- 选择凯雷德、萨博班、GMC 育空同级车型
- 填写机场接送、私人路线、隐私需求、航班号和联系方式
- 选择定金、信用卡预授权或全款
- 跳转 Stripe 完成付款/绑卡
- 通过 WhatsApp 私密确认订单

## 必须改的 4 个地方

打开 `index.html`，找到 `CONFIG`：

```js
const CONFIG = {
  whatsappNumber: "10000000000",
  stripeDepositLink: "https://buy.stripe.com/REPLACE_WITH_YOUR_DEPOSIT_LINK",
  stripeHoldLink: "https://buy.stripe.com/REPLACE_WITH_YOUR_CARD_AUTH_LINK",
  stripeFullPayLink: "https://buy.stripe.com/REPLACE_WITH_YOUR_FULL_PAYMENT_LINK"
};
```

改成：

- 你的 WhatsApp 电话号码
- Stripe 定金链接
- Stripe 信用卡预授权/绑卡链接
- Stripe 全款链接

## 上线方式

最快方式：

1. 打开 Netlify Drop
2. 上传整个 `private-driver-platform` 文件夹
3. 得到一个公网网址
4. 绑定你已经购买的域名：`booklaxdriver.com`

## 合规提醒

不要自己保存客户银行卡号。银行卡、信用卡、预授权和追讨费用应通过 Stripe、Square、PayPal 等合规支付平台完成。服务条款、取消费、等待费和空跑费建议由加州当地律师确认后上线。
